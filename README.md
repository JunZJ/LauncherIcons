# LauncherIcons
Repo for Pico and Oculus Launcher Icons

### Pico
- pico_landscape
- pico_square
### Oculus
- oculus_landscape
- oculus_square
- oculus_portrait
### Viveport
- viveport_small
- viveport_medium
- viveport_large
- viveport_square
